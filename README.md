dot-vim
=======

My Vim configuration files

--------------

## Install:

    cd ~
    git clone --recursive git@github.com:zenja/dot-vim.git
    cd dot-vim
    ln .vimrc ./.vimrc

or:

    cd ~
    git clone git@github.com:zenja/dot-vim.git
    cd dot-vim
    git submodule update --init --recursive
    ln .vimrc ./.vimrc
